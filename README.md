# CyberEDT Hash Generator

A modern, professional Hash Generator web tool for the CyberEDT platform.

## Features

## Features

*   **Full Hash Generation**: Support for **MD5, SHA-1, SHA-256, SHA-512, SHA-3**, and **Base64** encoding.
*   **File Hashing**: Upload files of any type to generate their checksums locally.
*   **Hash Comparison**: Built-in tool to instantly verify hash matches/mismatches.
*   **Real-time Mode**: Optional toggle to generate hashes instantly as you type.
*   **Theme Support**: Toggle between **Cyber Dark** and **Professional Light** modes.
*   **Privacy First**: 100% Client-side processing. No data leaves your browser.
*   **Educational**: Integrated guide on algorithms and security best practices.
*   **Export Options**: One-click Copy to Clipboard and Download as .txt.

## Tech Stack

*   **Vite**: Fast frontend build tool.
*   **Tailwind CSS**: Utility-first CSS framework for rapid styling.
*   **Vanilla JavaScript**: No heavy framework dependencies.

## Setup & Run

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Project Structure

*   `index.html`: Main entry point with the UI structure.
*   `src/`: Application source code (if you expand later).
*   `style.css`: Custom CSS and Tailwind directives.
*   `main.js`: Main JavaScript logic for UI interactions.
*   `tailwind.config.js`: Tailwind configuration.
